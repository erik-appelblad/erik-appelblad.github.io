The blog is a go!

Erik appelblad | fre 08 december 2017

So this is what the Python powered static site generator Pelican looks like with the attila theme. I like it, it looks crisp and clean and quite minimalistic. I'm planning to add more content later though. Setting up Pelican for use on GitHub Pages was quite simple using git in the terminal. Configuring everything and getting the theme to work was a bit of a pain though. I discovered some "bugs" that might indicate that the theme isn't updated to the latest version of Pelican, maybe I'll investigate this further and send a pull request. Now I'm looking forward to publish some content here.
